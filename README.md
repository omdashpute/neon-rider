# Neon Rider

Neon Rider is a fast-paced endless bike runner set in a neon-lit cyberpunk environment. The game focuses on smooth controls, progressive difficulty, and skill-based gameplay, combined with a calming lofi visual style.

## Overview

The player rides a futuristic bike through an endless highway filled with dynamic traffic and obstacles. The objective is to survive as long as possible while maximizing score through precise movement and risk-taking.


## Gameplay

* Swipe left or right to switch lanes
* Tap to activate boost

The game gradually increases in speed, requiring quick reactions and decision-making. Players are rewarded for close calls (near-misses), encouraging aggressive and skillful play.



## Features

* Endless procedural road generation
* Dynamic obstacle spawning
* Near-miss scoring system
* Progressive speed increase
* Coin collection mechanics
* Responsive touch controls



## Visual Style

Neon Rider follows a neon lofi cyberpunk aesthetic, featuring glowing environments, reflective surfaces, and a consistent purple-blue color palette to create an immersive yet relaxed atmosphere.



## Technology Stack

* Unity
* C#
* Blender
* Photoshop
* Audacity
* Git

## Vision

The goal of Neon Rider is to create a polished endless runner that balances engaging gameplay with a distinct visual identity.
